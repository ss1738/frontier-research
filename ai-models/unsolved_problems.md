# AI Models & Architecture — Frontier Unsolved Problems

## TOP 3 GAPS

### 1. Compositional Generalization — The Local-to-Global Synthesis Failure in Transformers
- 2025 study across 302 ARC tasks (arXiv:2512.07109): 69.5% of tasks achieve >80% cell-level accuracy yet <10% grid-level accuracy — the model solves subproblems but cannot compose the answer
- 35.3% of ARC-AGI-2 tasks exhibit "low neural affinity" for transformers (51.9% performance vs 77.7% on high-affinity tasks, p<0.001). This gap is structural, not a training-data problem
- Transformers with L layers are hardcapped at ~L parallelizable computation steps per forward pass — provable ceiling on deep compositional chains ("Shattered Compositionality," arXiv:2601.22510)
- The root mechanism is variable-binding failure: transformers implement approximate VSA unbinding via attention (queries = role spaces, values = fillers, weights = soft unbinding), but this is implicit and unstable (arXiv:2512.14709)
- Variables get confused across compositional chains longer than ~4 steps (arXiv:2505.20896)
- Proposed solutions — neuro-symbolic hybrids, Chain-of-Thought, test-time search — all work around the problem rather than fixing the architecture
- **Specific gap**: No architecture has been trained end-to-end with dedicated hyperdimensional binding layers between attention blocks plus an explicit role-filler separation loss. The VSA-transformer hybrid with binding heads is proposed in literature (arXiv:2512.14709) but has not been rigorously ablated at scale
- **Research unlock**: A transformer that passes compositional generalization benchmarks (SCAN, COGS, ARC-AGI-2) without test-time search would unblock reliable multi-step tool-use in agents and genuine program synthesis

### 2. Persistent Agent Memory — No Solved Episodic-to-Semantic Transition
- Core unsolved problem: the episodic-to-semantic transition — when and how do repeated episodic records graduate into reusable semantic knowledge? No system has a principled answer (arXiv:2603.07670, "Memory for Autonomous LLM Agents," 2026 survey)
- Neuroscience model: hippocampal sharp-wave ripple replay drives cortical consolidation, selectively strengthening high-importance traces. LLM analogs (SleepGate, SCM, LightMem, all 2025–2026) operate on KV-cache or external stores — NO system performs weight-level consolidation of agent experience offline
- Hoarding-vs-amnesia dilemma unresolved: aggressive compression loses rare critical facts silently ("memory blindness" — no error signal when important information is evicted). LoCoMo benchmark (300+ turn, 9k–16k token conversations): no current system handles importance-weighted retention reliably
- Reflection-based memory (MemGPT, Generative Agents) creates self-reinforcing error: false reflections persist without validation gates; confidence decay mechanisms absent from all deployed systems
- "Episodic Memory is the Missing Piece for Long-Term LLM Agents" (arXiv:2502.06975): identifies five required properties; field lacks a single benchmark testing all five simultaneously
- **Specific gap**: No system implements the full CLS loop for LLM agents: fast-path episodic store with automatic importance estimation → periodic offline weight-updating (LoRA-style gradient steps on agent's own replay) consolidating high-importance episodes into parametric memory → validation gate blocking reflection-derived updates that contradict grounded facts
- **Research unlock**: A genuine long-horizon agent that accumulates skill from experience across sessions without catastrophic forgetting and without unbounded context growth — prerequisite for any autonomous research or engineering agent operating over weeks

### 3. Model Collapse from Synthetic Data — The True Causal Mechanism Is Not Isolated
- Established phenomenon: recursive training on model-generated data degrades performance, destroys tail of learned distribution (Shumailov et al., 2024 Nature; Dohmatob et al., arXiv:2402.07712)
- Two competing mechanistic accounts, NOT experimentally disambiguated: (1) variance collapse — each generation compresses distribution, minority/tail data vanishes first; (2) population risk drift — generated distribution diverges from true distribution, rising population risk across recursive cycles (Bertrand 2024; Suresh 2024)
- Strong model collapse (ICLR 2025, arXiv:2410.04840): even 0.1% synthetic contamination asymptotically breaks neural scaling laws — larger datasets stop helping. But proof is in regression setting; causal mechanism at neural-network level not demonstrated
- Accumulation hypothesis (arXiv:2404.01413): collapse prevented by keeping all real data alongside synthetic — only delays as real-data fraction asymptotically shrinks
- Verifier-based filtering (arXiv:2510.16657): early gains then plateau or reversal unless verifier is perfectly reliable. Long-run convergence of filtered synthetic retraining uncharacterised theoretically
- **Specific gap**: No study has run a controlled causal intervention that independently manipulates (a) distributional shift of the synthetic generator, (b) tail frequency of synthetic samples, and (c) quality-filter threshold, with all other variables held constant, across multiple model scales. The field has correlational evidence but no factorial design that isolates the mechanism
- **Falsifiable hypothesis**: If variance collapse is primary (not distributional drift), then reweighting synthetic data by inverse-frequency of the sample in the generator's output distribution before retraining should substantially delay collapse independently of accumulation. Not tested
- **Research unlock**: Once causal mechanism isolated, determines how much synthetic data is safe to use in the post-internet-saturation training regime all frontier labs face by 2027

## WILD HYPOTHESIS
**Chronological Pretraining (CPT)**: Every LLM trains on a corpus shuffled uniformly at random across time — a 2003 Wikipedia article, a 2021 Reddit post, and a 2018 news story are fed in arbitrary order. The hypothesis: strict chronological ordering of documents by real-world publication date would produce qualitatively different internal representations — the model would learn causality, temporal precedence, and knowledge accumulation as structural priors rather than surface patterns. The first experiment nobody has run rigorously: pretrain two otherwise-identical models (same tokenizer, same data mix, same total tokens) where one receives documents in strict publication-date order (with temporal position encoded) and one receives standard shuffled order. Evaluate on: (1) temporal reasoning (TimeQA, TRAM, ChronoSense), (2) causal reasoning (BIG-Bench causal judgment), (3) knowledge cutoff sharpness (LLMLagBench), (4) in-context updating speed when contradicting old facts with newer ones. Closest work: curriculum-learning studies (arXiv:2601.21698) test linguistic difficulty ordering; two-phase pretraining (arXiv:2412.15285) does easy-then-hard; neither uses real-world temporal order. "Do LLMs Understand Chronology?" (arXiv:2511.14214) tests evaluation of chronological reasoning but not training. The specific ablation — graded temporal temperature in sampling distribution vs hard chronological curriculum vs random — has not been run.
